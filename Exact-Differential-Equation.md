# Exact Differential Equation
## General Form
```
f(x, y) =    dy/dx   = - (M(x, y)/N(x, y))
        => Mdx + Ndy = 0
```

## Criteria for Exactness
* The necessary and sufficient condition for `Mdx + Ndy = 0` to be exact is:
  ```
   𝛿M     𝛿N
  ---- = ----
   𝛿y     𝛿x
  ```
